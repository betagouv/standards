# Le produit suit ses bonnes pratiques sur DashLord

## Description

DashLord est un outil développé par la communauté beta.gouv.fr qui
permet d'effectuer des vérifications automatiques sur votre site web :
une fois l'URL de votre produit renseignée, DashLord va parcourir le
site et trouver, par exemple, la déclaration d'accessibilité, le temps
de chargement de la page, les problèmes de contraste, etc.

Cette première couverture de tests est une bonne pratique à adopter
pour assurer l'exemplarité de votre produit.

## Critères

- le produit est configuré sur le DashLord de son incubateur
